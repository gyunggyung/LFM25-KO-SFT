## 학습 설정

기본값:

| item | value |
|---|---:|
| base checkpoint | Stage2 `final_full` |
| max sequence | 8192 |
| per-device batch | 1 |
| gradient accumulation | 16 |
| effective batch | 128 sequences/update |
| learning rate | 1e-6 |
| save steps | 500 |
| target hub repo | `LLM-OS-Models/LFM2.5-8B-A1B-KO-Agentic-SFT` |

8192 context를 우선 사용한다. 16k/32k agentic long-context는 최종 quick 평가 후
별도 stage로 돌리는 것이 안전하다.
