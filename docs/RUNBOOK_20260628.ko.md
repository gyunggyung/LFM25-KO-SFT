# LFM2.5 KO SFT 실행 Runbook

작성 시각: 2026-06-28 19:20 KST

## 현재 우선순위

1. full SFT 학습 체인을 끊지 않는다.
2. Stage1 4k가 끝나면 Stage1 8k를 자동 시작한다.
3. Stage1 8k가 끝나면 Stage2 plus KoTSQA를 자동 시작한다.
4. Stage2가 끝나면 `base -> CPT -> SFT` 최종 vLLM/lm-eval queue를 자동 시작한다.
5. 외부 하네스 설치는 CPU/network 작업이라 학습과 병렬로 진행한다.
